## Краткое описание проекта

**SMM Automation API** — Laravel API для автоматизации задач SMM (Social Media Marketing), в основном для работы с Reddit.

### Основное назначение
Автоматизация регистрации и управления профилями в социальных сетях через браузерную автоматизацию.

### Ключевые компоненты

1. **ProfileBot** — управление профилями ботов:
   - Хранение данных (email, Reddit логины/пароли)
   - Статусы регистрации и cookies
   - Интеграция с AdsPower (браузерная автоматизация)
   - Привязка к прокси и спринтам

2. **TaskManager** — управление задачами:
   - Постановка задач на выполнение
   - Отслеживание статусов (pending, completed, failed)
   - Связь с профилями и сценариями

3. **Agent** — обработка ответов от браузерного агента:
   - `ProcessBrowserReply` — обработка результатов браузерных задач
   - `ProcessCookieReply` — обработка cookie-задач
   - Обновление статусов профилей на основе результатов

4. **RabbitMQ** — асинхронная обработка:
   - Получение результатов от браузерного агента
   - Обработка через очереди
   - Использование correlation_id для связи задач

5. **AdsPower** — интеграция с AdsPower API:
   - Создание/обновление профилей браузеров
   - Управление группами профилей

6. **Scenario** — сценарии автоматизации (например, регистрация на Reddit)

7. **Дополнительные модули**:
   - **Sprint** — организация работы по спринтам
   - **Proxy** — управление прокси-серверами
   - **Tag** — тегирование профилей
   - **Comment** — обработка комментариев

### Архитектура
- Модульная структура: каждый сервис в отдельной папке с собственным ServiceProvider
- Использование контрактов (интерфейсов) для зависимостей
- Repository pattern для работы с данными
- JWT-аутентификация
- Очереди через RabbitMQ для асинхронной обработки

### Типичный workflow
1. Создание ProfileBot с данными для регистрации
2. Постановка задачи через TaskManager на выполнение сценария
3. Задача отправляется в RabbitMQ очередь
4. Браузерный агент (AdsPower) выполняет задачу
5. Результат возвращается через RabbitMQ
6. Agent обрабатывает результат и обновляет статусы

Проект предназначен для масштабируемой автоматизации SMM-задач с разделением ответственности между компонентами.
